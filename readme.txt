Portfolio Website

Overview

This is a portfolio website developed as a freelance project to enhance web development skills. The website showcases various sections, including an AI-powered chatbot integrated using the Tawk API.

Features

Responsive Design: The website is fully responsive and adapts to different screen sizes.

Portfolio Showcase: Displays projects and achievements effectively.

AI Chatbot: Integrated using the Tawk API for interactive user engagement.

Smooth Navigation: Easy-to-use interface with seamless navigation.

SEO Friendly: Optimized for search engines to improve visibility.
